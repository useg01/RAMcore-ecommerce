# RAMcore E-commerce - Angular Avanzado

Proyecto desarrollado para el curso de Javascript Avanzado.

## Tecnologías
- Angular 18 (Standalone Components, Routing, Services)
- Bootstrap 5 (Responsive Design)
- JSON Server (API REST simulada)
- LocalStorage para seguridad de rutas

## Instalación y Uso Local
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Levantar la base de datos: `npx json-server --watch db.json`.
4. Iniciar el proyecto: `ng serve`.
